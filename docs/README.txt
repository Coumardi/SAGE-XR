Refer to here for all building information and structure questions.

https://www.sphinx-doc.org/en/master/usage/restructuredtext/basics.html

For Structure levels use the below notation.
# with overline, for parts

* with overline, for chapters

= for sections

- for subsections

^ for subsubsections

" for paragraphs

run the make_html file to build the document.
documents will be in the build folder when completed.
